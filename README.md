## 🚀 Overview
A RESTful backend service for managing student details. Supports adding, updating, deleting, and retrieving student information. Built with Spring Boot and JPA for robust data persistence.

## ✨ Key Features
- **Add Student** (Create new student records).
- **View All Students** (Retrieve list of all students).
- **Update Student** (Modify existing student details).
- **Delete Student** (Remove student records).
- **Search Student** (Find students by name or ID).

## 🛠️ Tech Stack
- **Backend**: Java 8, Spring Boot 2.7.16, Spring Data JPA, Hibernate.
- **Database**: MySQL 8.0.
- **Build Tool**: Maven.
